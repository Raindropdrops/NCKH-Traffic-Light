# User Manual - Traffic Light Dashboard

> **Status:** Placeholder - WP5.3
> **Depends on:** WP2.* Node-RED Dashboard

## Accessing the Dashboard

1. Open browser: <http://localhost:1880/ui>
2. Dashboard should show Traffic Light control panel

## Dashboard Sections

### Status Panel

- Current mode (AUTO/MANUAL/BLINK/OFF)
- Current phase (0-5)
- Uptime
- Online/Offline indicator

### Control Panel

- Mode buttons: AUTO, MANUAL, BLINK, OFF
- Phase buttons (in MANUAL mode): 0-5
- EMERGENCY button

### Traffic Light Visualization

- Real-time LED state display
- Direction A (North-South)
- Direction B (East-West)

### Telemetry Charts

- WiFi RSSI over time
- Free heap memory over time
